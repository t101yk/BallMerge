Frequently Asked Questions

How do I back up my game data? Will I lose it?
All game data (progress, scores, settings) is stored only locally on your device. Uninstalling the app will clear all data. There is currently no cloud backup feature. If you need to reset the game, you can use the "Reset Game" option within the app.
Will the game cost money? Are there ads?
Currently completely free – no ads, no in-app purchases. If paid features are introduced in the future, we will notify users in advance through App Store update descriptions and in-app notifications.
What should I do if the game is laggy or crashes?
Please try the following steps:
1. Close other apps running in the background
2. Restart your device
3. Make sure your iOS is updated to the latest version
4. If the issue persists, please contact us via email with your device model and iOS version
How can I contact the developer?
You can reach us through:
• Email: fhljx120@gmail.com
• GitHub: github.com/t101yk/BallMerge
We will get back to you as soon as possible.
Which languages are supported?
Currently supported: Simplified Chinese, English, Japanese. More languages will be added in future updates.
